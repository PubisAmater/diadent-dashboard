# Бриф: 3 варианта дашборда Диадент

## Заказчик
Владимир Горнак — владелец группы стоматологических клиник "Диадент"

## Требования
- **Mobile-first** (телефон приоритет, но и комп тоже). Лучше две отдельные версии.
- **Роли доступа:**
  - Владимир (админ): полный доступ + редактирование + админ-панель
  - Фин.директор, опер.директор, директор по медицине: полный просмотр
  - Топ-менеджмент отделов: только свои отделы
- **Фокус:** ключевые показатели без перегруза, но с глубиной по клику
- **Рекомендации:** подсветка проблем, рекомендации по улучшению
- **Данные:** раз в день + возможность реального времени
- **Источники:** Google Sheets ДДС + DentalPro API (врачи, визиты, загрузка)
- **Без тёмной темы**

## Структура бизнеса
- 5 клиник: Светлановский (ДИАНА), Серпуховская (ДИАДЕНТ), Просвещения (ДИАДЕНТ53), Бухарестская (ДИАДЕНТ110), Энгельса (ДИАДЕНТ124)
- ЗТЛ — зуботехническая лаборатория
- УЦ — учебный центр
- Направления: терапия, хирургия, имплантология, ортодонтия, протезирование, пародонтология, детская стомат.

## Ключевые KPI
- Выручка (группа, филиал, врач, направление)
- Расходы по статьям (ФОТ, материалы, аренда, налоги и т.д.)
- Прибыль / маржа
- EBITDA
- Загрузка кресел
- Средний чек (общий, по направлению)
- Количество визитов
- Средний чек per visit
- NPS / качество
- Дебиторка
- План/факт

## 3 концепции

### Вариант A: "Command Center" (Командный центр)
- Один экран = полная картина бизнеса
- Dashboard-of-dashboards: карточки с ключевыми метриками
- Tap на карточку → drill-down в детали
- Стиль: строгий, информационный, как Bloomberg terminal но для стоматологии
- Мобилка: карточки стакаются, swipe между разделами

### Вариант B: "Story Flow" (Поток историй)
- Дашборд как лента новостей/историй
- Каждый блок = инсайт или рекомендация
- "Выручка выросла на 19% — вот почему" с drill-down
- AI-рекомендации встроены прямо в карточки
- Стиль: modern, как Notion/Linear — минималистичный
- Мобилка: вертикальный скролл, как Instagram stories для бизнеса

### Вариант C: "Cockpit" (Кокпит)
- Навигация по уровням: Группа → Филиал → Отдел → Врач
- Каждый уровень = свой полноценный дашборд
- Сравнение "бок о бок" (филиал vs филиал, врач vs врач)
- Стиль: app-like, как Revolut/банковское приложение
- Мобилка: tab bar + cards, native app feeling
